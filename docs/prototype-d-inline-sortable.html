Prototype D

Inline Editing + Sortable Columns + Rule Engine

  • Emoji prefix on every category for quick scanning
  • Uncategorized transactions highlighted with yellow box
  • Click any category cell to assign or change category via 2-level dropdown
  • Rules match exact transaction text and apply to all identical transactions
  • Sort by Kategori to group uncategorized for bulk editing
  • Rules persist in localStorage across sessions

Pros vs others

  • Sort by Kategori groups uncategorized rows for fast bulk editing (unique)
  • Rules persist across sessions via localStorage (unique)
  • Emoji prefixes aid quick visual scanning
  • Combines inline editing (like A) with workflow tooling

Cons vs others

  • Extra concept to learn (rules) — higher cognitive load
  • No searchable picker (C has a combobox)
  • No dedicated bulk panel (B has one)
  • No explicit undo (C has a toast)
